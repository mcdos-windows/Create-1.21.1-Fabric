## 🇨🇳 中文介绍

### 项目简介

本仓库是 Create（机械动力）玩家自制的非官方移植版本,由mcdos-windows与Cao123创建。

源码基于官方 Create Fabric 1\.20\.1 版本，向上升级适配，完美兼容 Minecraft 1\.21\.1 Fabric 加载器。

官方机械动力仅为 1\.21\.1 提供 NeoForge 版本，**无官方 Fabric 构建**。

本项目与 Create Fly 无任何关联，仅基于官方旧版 1\.20\.1 Fabric 源码进行版本移植适配。
